## Required API Endpoints

### Expenses
- `GET /expenses` - Returns all expenses
- `POST /expenses` - Creates new expense
- `PUT /expenses/:id` - Updates existing expense
- `DELETE /expenses/:id` - Removes expense

### Categories
- `GET /categories` - Returns all categories
